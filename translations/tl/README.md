# IoT para sa mga Nagsisimula - Isang Kurikulum

Ang mga Azure Cloud Advocates sa Microsoft ay natutuwa na mag-alok ng 12-linggong, 24-leksyon na kurikulum tungkol sa mga batayan ng IoT. Bawat leksyon ay may kasamang pre- at post-lesson quizzes, mga nakasulat na tagubilin para tapusin ang leksyon, solusyon, asignatura at iba pa. Ang aming pedagohiyang naka-project ay nagpapahintulot sa iyo na matuto habang bumubuo, isang patunay na paraan para ang mga bagong kasanayan ay manatag.

Sinasaklaw ng mga proyekto ang paglalakbay ng pagkain mula bukid hanggang mesa. Kasama dito ang pagsasaka, logistika, pagmamanupaktura, retail at consumer - lahat ay mga sikat na industriya para sa mga IoT device.

## Pedagohiya

Pinili namin ang dalawang prinsipyong pedagogical habang binubuo ang kurikulum na ito: tiyakin na ito ay naka-project at may mga madalas na quiz. Sa pagtatapos ng seryeng ito, magkakaroon ang mga mag-aaral ng monitoring at watering system para sa halaman, vehicle tracker, smart factory setup para subaybayan at suriin ang pagkain, at voice-controlled cooking timer, at matutunan ang mga batayan ng Internet of Things kabilang ang pagsulat ng code para sa device, pagkonekta sa cloud, pagsusuri ng telemetry at pagpapatakbo ng AI sa gilid.

Sa pamamagitan ng pagtiyak na ang nilalaman ay tumutugma sa mga proyekto, mas ginagawang kaakit-akit ang proseso para sa mga mag-aaral at pinapalawak ang retention ng mga konsepto.

Dagdag pa, ang mababang-panisik na quiz bago ang klase ay nagtatalaga ng layunin ng mag-aaral patungo sa pag-aaral ng isang paksa, habang ang pangalawang quiz pagkatapos ng klase ay nagtitiyak ng karagdagang retention. Ang kurikulum na ito ay dinisenyo upang maging flexible at masaya at maaaring kunin nang buo o bahagya. Nagsisimula ng maliit ang mga proyekto at nagiging mas kumplikado sa pagtatapos ng 12-linggong siklo.

Bawat proyekto ay nakabase sa totoong hardware na available sa mga estudyante at hobbyists. Tinitingnan ng bawat proyekto ang espesipikong domain ng proyekto, nagbibigay ng kaugnay na background na kaalaman. Para maging matagumpay na developer, nakatutulong na maunawaan ang domain kung saan nilulutas ang mga problema, ang pagbibigay ng background knowledge na ito ay nagpapahintulot sa mga mag-aaral na pag-isipan ang kanilang mga solusyon sa IoT at mga natutunan sa konteksto ng uri ng real-world na problema na maaaring hingin sa kanila bilang isang IoT developer. Natutunan ng mga mag-aaral ang 'bakit' ng solusyong kanilang binubuo, at nakakakuha ng pagpapahalaga sa end user.

## Hardware
Mayroon tayong dalawang pagpipilian ng IoT hardware na magagamit para sa mga proyekto depende sa personal na kagustuhan, kaalaman sa programming language o mga kagustuhan, mga layunin sa pag-aaral at availability. Nagbigay din kami ng 'virtual hardware' na bersyon para sa mga walang access sa hardware, o nais matuto nang higit pa bago magdesisyon sa pagbili. Maaari kang magbasa pa at makakita ng 'shopping list' sa [hardware page](./hardware.md), kasama ang mga link para bumili ng kumpletong kits mula sa aming mga kaibigan sa Seeed Studio.
