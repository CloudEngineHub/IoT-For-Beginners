# ஆரம்பิกருக்கான IoT - ஒரு பாடத்திட்டம்

Microsoft இல் Azure Cloud வழிகாட்டிகள் 12 வாரம், 24 பாடங்கள் கொண்ட IoT அடிப்படைகள் பற்றிய பாடத்திட்டத்தை வழங்க மகிழ்ச்சியடைகின்றனர். ஒவ்வொரு பாடமும் முன் மற்றும் பின் பாடத் தேர்வுகள், கையேடு, தீர்வு, பணிமுறை மற்றும் அதிகம் உட்படுகிறது. நமது திட்ட அடிப்படையிலான கற்றல் முறையின் மூலம் நீங்கள் உருவாக்கின போது கற்றுக்கொள்ள முடியும், இது புதிய திறன்கள் 'உட்கார' உதவும் என்பது நிரூபிக்கப்பட்டது.

இந்த திட்டங்கள் உணவு பயிர் முதல் மேஜை வரை பயணத்தை கையாள்கின்றன. இதில் விவசாயம், போக்குவரத்து, உற்பத்தி, விற்பனை மற்றும் நுகர்வோர் ஆகியவை உள்ளன - அனைத்தும் IoT சாதனங்களுக்கான பிரபலமான தொழில்துறை பகுதிகள்.

## கற்றல் முறை

இந்த பாடத்திட்டத்தை உருவாக்கும் போது நாம் இரண்டு கற்றல் கொள்கைகளைத் தேர்ந்தெடுத்துள்ளோம்: திட்ட அடிப்படையிலானது மற்றும் அடிக்கடி நடைபெறும் தேர்வுகள் அடங்கியது. இந்த தொடரின் முடிவில், மாணவர்கள் செடியின் கண்காணிப்பு மற்றும் நீர்ப்பாசனம், வாகன கண்காணிப்பு, உணவை கண்காணிக்கும் புத்திசாலி தொழிற்சாலை அமைப்பு மற்றும் குரல் கட்டுப்பாட்டில் இயங்கும் சமையல் நேரக் கணக்குப்படுத்தியை உருவாக்கப்படும், மேலும் Internet of Things நற்குணங்களை தெரிந்து, சாதனக் குறியீடு எழுத, மேகத்துடன் இணைப்பு, தொலைநிலை தரவுகளை பகுப்பாய்வு மற்றும் எட்வை AI இயக்கு கற்றுக்கொள்வார்கள்.

பாட உள்ளடக்கம் திட்டங்களுக்கு இணங்க இருப்பதால் மாணவர்களுக்கு கற்றல் விருப்பமும், கருத்துகளை நினைவில் வைக்கும் திறனும் மேம்படும்.

மேலும், வகுப்புக்கு முன்னதாக குறைந்த மதிப்பெண் தேர்வு மாணவரின் கற்றல் நோக்கத்தை ஏற்படுத்தும், வகுப்பிற்குப் பிறகு இரண்டாவது தேர்வு கூடுதல் நினைவாற்றலை உறுதி செய்யும். இந்த பாடத்திட்டம் நெகிழ்வுள்ள மற்றும் வினோதமானது, முழுமையாகவோ, ஒரு பகுதியை மட்டும் எடுத்துக் கொண்டாலும் பயிற்றுவிக்கலாம். 12 வார காலத்துக்குள் திட்டங்கள் சிறியதாக துவங்கி, சிக்கலானதாக மாறும்.

ஒவ்வொரு திட்டமும் மாணவர்களுக்கும் ஆர்வலர்களுக்கும் கிடைக்கும் உண்மையான கருவிகளின் சுற்றிலும் உள்ளது. ஒவ்வொரு திட்டமும் குறிப்பிட்டத் துறையை ஆராய்ந்து, பொருத்தமான பின்னணி அறிவைப் வழங்குகிறது. துறையைப் புரிந்துகொள்ளும் திறன் ஒரு வெற்றிகரமான டெவலப்பராக இருக்க உதவும்; இது மாணவர்களுக்கு IoT தீர்வுகளை மற்றும் கற்றல்களை உண்மையான உலக பிரச்சனைகளுடன் தொடர்புபடுத்தி யோசிக்க உதவும். மாணவர்கள் அவர்கள் உருவாக்கும் தீர்வுகளின் காரணத்தைப் புரிந்து கொண்டு இறுதி பயனாளரிடம் பாராட்டை பெறுவார்கள்.

## ஹார்ட்வேர்
நாம் விருப்பப்படி, நிரல் மொழி அறிவு அல்லது விருப்பங்கள், கற்றல் இலக்குகள் மற்றும் கிடைக்கும் வசதிகளின்படி, திட்டங்களுக்கு பயன்படுத்த இரண்டு வகை IoT ஹார்ட்வேர்களை கொண்டிருக்கிறோம். ஹார்ட்வேருக்கு அணுகல் இல்லாதோர் அல்லது வாங்க முன் மேலும் கற்றுக்கொள்ள விரும்புவோருக்காக 'மெய்நிகர் ஹார்ட்வேர்ல்' பதிப்பையும் வழங்கியுள்ளோம். முழு கிட்களை எங்களுடைய நண்பர்கள் Seeed Studio இலிருந்து வாங்க இணைப்புகளுடன் சேர்த்து, [hardware page](./hardware.md) இல் 'ஷாப்பிங் பட்டியல்' ஐ வாசிக்கலாம்.

## ஆஃப்லைன் அணுகல்

[Docsify](https://docsify.js.org/#/) ஐ பயன்படுத்தி இந்த ஆவணத்தை ஆஃப்லைனிலும் இயக்க முடியும். இந்த ரெப்போவை ஃபோর্ক் செய்து, உங்கள் உள்ளூரில் [Docsify ஐ நிறுவி](https://docsify.js.org/#/quickstart) இந்த ரெப்போவின் ரூட் கோப்பகத்தில் `docsify serve` என தட்டச்சு செய்யவும். இந்த இணையதளம் உங்கள் localhost இல் போர்ட் 3000 இல் சேவை செய்யப்படும்: `localhost:3000`.

## வினாடி விடை

ஒவ்வொரு அத்தியாயத்திலும் உங்கள் அறிவை சோதிக்கும் இடைத்தேர்வு வினாத்தாளுக்கு சமூகத்துக்கு நன்றி. உங்கள் அறிவை [இங்கே](https://ff-quizzes.netlify.app/en/) சோதிக்கலாம்.

### PDF

பொருந்தியிருக்காவிட்டால் ஆஃப்லைன் அணுகலைக் கொண்டு இந்த உள்ளடக்கத்தின் PDF உருவாக்கலாம். இதற்கு, [npm நிறுவியிருப்பதை](https://docs.npmjs.com/downloading-and-installing-node-js-and-npm) உறுதி செய்து, இந்த ரெப்போவின் ரூட் கோப்பகத்தில் கீழ்க்கண்ட கட்டளைகளை நடைமுறைபடுத்தவும்:

```sh
npm i
npm run convert
```
