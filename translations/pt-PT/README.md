# IoT para Iniciantes - Um Currículo

Os Cloud Advocates da Azure na Microsoft têm o prazer de oferecer um currículo de 12 semanas com 24 lições sobre os fundamentos do IoT. Cada lição inclui testes pré e pós-lição, instruções escritas para completar a lição, uma solução, uma tarefa e mais. A nossa pedagogia baseada em projetos permite-lhe aprender construindo, uma forma comprovada de fixar novas competências.

Os projetos abrangem a jornada dos alimentos desde a quinta até à mesa. Isto inclui agricultura, logística, fabrico, retalho e consumidor - todas áreas industriais populares para dispositivos IoT.

## Pedagogia

Escolhemos dois princípios pedagógicos ao construir este currículo: garantir que é baseado em projetos e que inclui quizzes frequentes. No final desta série, os estudantes terão construído um sistema de monitorização e rega de plantas, um rastreador de veículos, uma instalação de fábrica inteligente para rastrear e verificar alimentos, e um temporizador de cozinha controlado por voz, e terão aprendido os fundamentos do Internet das Coisas, incluindo como escrever código para dispositivos, conectar à cloud, analisar telemetria e executar IA na borda.

Garantindo que o conteúdo está alinhado com projetos, o processo torna-se mais envolvente para os estudantes e a retenção dos conceitos será aumentada.

Além disso, um quiz de baixo impacto antes da aula define a intenção do estudante para aprender um tópico, enquanto um segundo quiz após a aula assegura maior retenção. Este currículo foi desenhado para ser flexível e divertido, podendo ser feito na totalidade ou em parte. Os projetos começam pequenos e tornam-se cada vez mais complexos até ao final do ciclo de 12 semanas.

Cada projeto é baseado em hardware do mundo real disponível para estudantes e entusiastas. Cada projeto analisa o domínio específico do projeto, fornecendo conhecimentos de base relevantes. Para ser um programador bem-sucedido, ajuda compreender o domínio em que está a resolver problemas; fornecer este conhecimento de base permite que os estudantes pensem nas suas soluções e aprendizagens de IoT no contexto do tipo de problema real que poderão ser solicitados a resolver como desenvolvedores IoT. Os estudantes aprendem o 'porquê' das soluções que estão a construir, e ganham uma apreciação pelo utilizador final.

## Hardware
Temos duas opções de hardware IoT para usar nos projetos, dependendo da preferência pessoal, do conhecimento ou preferências da linguagem de programação, objetivos de aprendizagem e disponibilidade. Também fornecemos uma versão de 'hardware virtual' para quem não tem acesso ao hardware ou quer aprender mais antes de se comprometer com uma compra. Pode ler mais e encontrar uma 'lista de compras' na [página de hardware](./hardware.md), incluindo ligações para comprar kits completos dos nossos amigos da Seeed Studio.

## Cada lição inclui:

- esboço gráfico
- vídeo suplementar opcional
- quiz de preparação pré-lição
- lição escrita
- para lições baseadas em projetos, guias passo a passo para construir o projeto
- verificações de conhecimento
- um desafio
- leitura suplementar
- tarefa
- [quiz pós-lição](https://ff-quizzes.netlify.app/en/)

> **Uma nota sobre os quizzes**: Todos os quizzes estão contidos na pasta quiz-app, totalizando 48 quizzes de três perguntas cada. Eles estão ligados a partir das lições mas a aplicação dos quizzes pode ser executada localmente ou implantada no Azure; siga as instruções na pasta `quiz-app`. Estão a ser gradualmente localizados.
