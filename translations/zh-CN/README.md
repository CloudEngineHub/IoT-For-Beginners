# 初学者物联网课程大纲

微软 Azure 云推广团队高兴地提供一个为期 12 周、共 24 课的关于物联网基础知识的课程。每节课包括课前和课后小测、书面指导以完成课程、解决方案、作业等内容。我们的项目驱动教学法让你在动手构建的过程中学习，这是一种经过验证的技能“牢固掌握”方法。

这些项目涵盖了从农场到餐桌的食物流通过程，包括农业、物流、制造、零售和消费者——这些都是物联网设备广泛应用的热门行业。

## 教学法

我们设计本课程时选择了两个教学原则：确保课程项目驱动和包含频繁的小测。学习完成本系列课程后，学生将构建一个植物监测和灌溉系统、车辆跟踪器、一个智能工厂用于追踪和检查食品，以及一个语音控制的烹饪定时器，并学习物联网基础知识，包括如何编写设备代码、连接云服务、分析遥测数据以及在边缘运行人工智能。

通过确保内容与项目紧密结合，学生学习过程更加投入，概念掌握更牢固。

此外，课堂前的小测设定学习者的学习目标，课堂后的小测则帮助进一步巩固知识。此课程设计灵活有趣，可整套学习，也可部分学习。项目从简单渐进到复杂，贯穿整个 12 周周期。

每个项目基于学生和爱好者能够获得的真实硬件。每个项目都会探讨特定领域，提供相关背景知识。成为成功的开发者有助于理解所解决问题的领域背景，提供这些背景让学生能在实际物联网问题的上下文中思考解决方案和学习内容。学生理解他们构建解决方案的“原因”，并体会到最终用户的需求。
