# IoT 初學者課程

Microsoft 的 Azure 雲端倡導者很高興提供一個為期 12 週、包含 24 課的物聯網（IoT）基礎課程。每課包括課前與課後測驗、完成課程的書面指示、解答、作業等。我們採取以項目為基礎的教學法，讓你邊做邊學，這是確保新技能穩固掌握的有效方法。

這些項目涵蓋食物從農場到餐桌的整個旅程。涵蓋農業、物流、生產、零售與消費者等領域，皆是物聯網裝置的重要應用產業。

## 教學方法

本課程採取兩大教學原則：以專案為基礎，以及包含頻繁的測驗。在這系列課程結束時，學生將完成一套植物監測與澆水系統、車輛追蹤器、智慧工廠設定用於追蹤與檢查食物，以及語音控制的烹飪計時器，並且學會物聯網基礎，包括撰寫裝置程式碼、連接至雲端、分析遙測資料及於邊緣運行 AI。

確保內容與專案對應，過程對學生更具吸引力，使得概念更易記憶。

此外，課前一個低壓力的測驗能讓學生設定學習目標，課後第二個測驗則促進記憶鞏固。本課程設計靈活且有趣，既可完整修習，也可選修部分。專案規模由淺入深，12 週結束時漸趨複雜。

每個專案都基於學生及興趣者可取得的實體硬體。專案會涵蓋特定領域，提供相關背景知識。成為成功的開發者，了解問題領域有幫助，這背景知識讓學生在思考物聯網解決方案及學習時，能根據真實問題情境設計。學生理解他們打造方案的「原因」，並能體會終端使用者的需求。
