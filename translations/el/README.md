# IoT για Αρχάριους - Ένα Αναλυτικό Πρόγραμμα Σπουδών

Οι υποστηρικτές του Azure Cloud στη Microsoft είναι στην ευχάριστη θέση να προσφέρουν ένα πρόγραμμα σπουδών 12 εβδομάδων, 24 μαθημάτων για τα βασικά του IoT. Κάθε μάθημα περιλαμβάνει κουίζ πριν και μετά το μάθημα, γραπτές οδηγίες για την ολοκλήρωση του μαθήματος, μια λύση, μια ανάθεση και άλλα. Η παιδαγωγική μας που βασίζεται σε έργα επιτρέπει να μαθαίνετε ενώ κατασκευάζετε, ένας αποδεδειγμένος τρόπος ώστε οι νέες δεξιότητες να «εδραιώνονται».

Τα έργα καλύπτουν το ταξίδι του φαγητού από το αγρόκτημα στο τραπέζι. Αυτό περιλαμβάνει αγροτική παραγωγή, logistics, κατασκευή, λιανική και καταναλωτές - όλα δημοφιλείς τομείς της βιομηχανίας για συσκευές IoT.

## Παιδαγωγική

Έχουμε επιλέξει δύο παιδαγωγικές αρχές κατά την κατασκευή αυτού του προγράμματος σπουδών: να είναι βασισμένο σε έργα και να περιλαμβάνει συχνά κουίζ. Μέχρι το τέλος αυτής της σειράς, οι μαθητές θα έχουν φτιάξει ένα σύστημα παρακολούθησης και ποτίσματος φυτών, μια συσκευή εντοπισμού οχημάτων, μια έξυπνη εγκατάσταση κατασκευής για την παρακολούθηση και τον έλεγχο φαγητού, και έναν χρονοδιακόπτη μαγειρικής που ελέγχεται φωνητικά, ενώ θα έχουν μάθει τα βασικά του Διαδικτύου των Πραγμάτων, συμπεριλαμβανομένου του πώς να γράφουν κώδικα για συσκευές, να συνδέονται στο cloud, να αναλύουν τηλεμετρία και να τρέχουν AI στην άκρη.

Με το να ευθυγραμμιστεί το περιεχόμενο με έργα, η διαδικασία γίνεται πιο ελκυστική για τους μαθητές και η διατήρηση των εννοιών θα ενισχυθεί.

Επιπλέον, ένα κουίζ χαμηλής πίεσης πριν από το μάθημα θέτει την πρόθεση του μαθητή να μάθει ένα θέμα, ενώ ένα δεύτερο κουίζ μετά το μάθημα εξασφαλίζει περαιτέρω διατήρηση. Αυτό το πρόγραμμα σπουδών σχεδιάστηκε για να είναι ευέλικτο και διασκεδαστικό και μπορεί να γίνει ολόκληρο ή μερικώς. Τα έργα ξεκινούν μικρά και γίνονται όλο και πιο σύνθετα μέχρι το τέλος του δωδεκάμηνου κύκλου.

Κάθε έργο βασίζεται σε πραγματικό υλικό διαθέσιμο σε μαθητές και χομπίστες. Κάθε έργο εξετάζει τον συγκεκριμένο τομέα του έργου, παρέχοντας σχετική βασική γνώση. Για να γίνει κάποιος επιτυχημένος προγραμματιστής, βοηθάει να κατανοεί τον τομέα στον οποίο επιλύει προβλήματα, και αυτή η βασική γνώση επιτρέπει στους μαθητές να σκέφτονται τις λύσεις και τις γνώσεις τους για το IoT στο πλαίσιο του είδους του πραγματικού προβλήματος που μπορεί να κληθούν να λύσουν ως προγραμματιστές IoT. Οι μαθητές μαθαίνουν το «γιατί» των λύσεων που κατασκευάζουν και αποκτούν εκτίμηση για τον τελικό χρήστη.

## Υλικό (Hardware)
Έχουμε δύο επιλογές υλικού IoT για χρήση στα έργα, ανάλογα με τις προσωπικές προτιμήσεις, τη γνώση της γλώσσας προγραμματισμού ή τις προτιμήσεις, τους στόχους μάθησης και τη διαθεσιμότητα. Έχουμε επίσης παράσχει μια έκδοση 'εικονικού υλικού' για όσους δεν έχουν πρόσβαση σε υλικό ή θέλουν να μάθουν περισσότερα πριν δεσμευτούν σε μια αγορά. Μπορείτε να διαβάσετε περισσότερα και να βρείτε μια 'λίστα αγορών' στη [σελίδα υλικού](./hardware.md), συμπεριλαμβανομένων συνδέσμων για να αγοράσετε ολοκληρωμένα κιτ από τους φίλους μας στο Seeed Studio.

## Κάθε μάθημα περιλαμβάνει:

- σημειώσεις σκίτσου
- προαιρετικό συμπληρωματικό βίντεο
- quiz προθέρμανσης πριν το μάθημα
- γραπτό μάθημα
- για μαθήματα βασισμένα σε έργα, οδηγίες βήμα προς βήμα για την κατασκευή του έργου
- ελέγχους γνώσης
- μια πρόκληση
- συμπληρωματική ανάγνωση
- εργασία
- [quiz μετά το μάθημα](https://ff-quizzes.netlify.app/en/)

> **Μια σημείωση σχετικά με τα κουίζ**: Όλα τα κουίζ περιέχονται στον φάκελο quiz-app, για συνολικά 48 κουίζ των τριών ερωτήσεων το καθένα. Συνδέονται από τα μαθήματα, αλλά η εφαρμογή κουίζ μπορεί να τρέξει τοπικά ή να αναπτυχθεί στο Azure· ακολουθήστε τις οδηγίες στο φάκελο `quiz-app`. Σταδιακά μεταφράζονται.
