# IoT para Iniciantes - Um Currículo

Advogados do Azure Cloud na Microsoft têm o prazer de oferecer um currículo de 12 semanas e 24 lições sobre os fundamentos de IoT. Cada lição inclui quizzes pré e pós-lição, instruções escritas para completar a lição, uma solução, uma tarefa e mais. Nossa pedagogia baseada em projetos permite que você aprenda enquanto constrói, uma maneira comprovada para novas habilidades 'fixarem'.

Os projetos cobrem a jornada do alimento da fazenda até a mesa. Isso inclui agricultura, logística, manufatura, varejo e consumidor - todas áreas populares da indústria para dispositivos IoT.

## Pedagogia

Escolhemos dois princípios pedagógicos ao construir este currículo: garantir que seja baseado em projetos e que inclua quizzes frequentes. Ao final desta série, os estudantes terão construído um sistema de monitoramento e rega de plantas, um rastreador de veículos, uma fábrica inteligente para acompanhar e verificar alimentos, e um temporizador de cozinha controlado por voz, além de terem aprendido o básico da Internet das Coisas, incluindo como programar dispositivos, conectar-se à nuvem, analisar telemetria e executar IA na borda.

Ao garantir que o conteúdo esteja alinhado com projetos, o processo se torna mais envolvente para os estudantes e a retenção dos conceitos será aumentada.

Além disso, um quiz de baixa pressão antes da aula define a intenção do estudante para aprender o tópico, enquanto um segundo quiz após a aula assegura retenção adicional. Este currículo foi projetado para ser flexível e divertido e pode ser realizado integralmente ou em partes. Os projetos começam pequenos e se tornam progressivamente mais complexos até o final do ciclo de 12 semanas.

Cada projeto é baseado em hardware do mundo real disponível para estudantes e entusiastas. Cada projeto examina o domínio específico do projeto, fornecendo conhecimento de base relevante. Para ser um desenvolvedor bem-sucedido, ajuda entender o domínio onde você está resolvendo problemas; fornecer esse conhecimento de base permite que os estudantes pensem sobre suas soluções e aprendizados de IoT no contexto do tipo de problema do mundo real que podem ser chamados para resolver como desenvolvedores IoT. Os estudantes aprendem o 'porquê' das soluções que estão construindo e ganham apreço pelo usuário final.

## Hardware
Temos duas opções de hardware IoT para usar nos projetos, dependendo da preferência pessoal, conhecimento ou preferência de linguagem de programação, objetivos de aprendizado e disponibilidade. Também fornecemos uma versão de 'hardware virtual' para aqueles que não têm acesso ao hardware ou desejam aprender mais antes de se comprometerem com uma compra. Você pode ler mais e encontrar uma 'lista de compras' na [página de hardware](./hardware.md), incluindo links para comprar kits completos com nossos amigos da Seeed Studio.
